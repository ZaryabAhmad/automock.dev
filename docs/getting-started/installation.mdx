# Installation

## Quick Install

To fully integrate Automock into your testing and dependency injection  frameworks, **you'll need to install two
packages: the Automock package for your chosen testing framework and the corresponding DI framework adapter.**

For **Jest**:

```bash npm2yarn
npm i -D @automock/jest
```

For **Sinon**:

```bash npm2yarn
npm i -D @automock/sinon
```

**And for your DI framework, choose the appropriate Automock adapter:**

## Seamless Integration with Different DI Frameworks

Automock extends its capabilities to integrate with various dependency injection  frameworks, such as NestJS and
InversifyJS, among others. Detailed insights into Automock's adaptability across frameworks are provided on our
dedicated [Adapters](/docs/adapters/intro) page.
